ü™ü Windows 11: Microsoft Family Safety

English Instructions

  1. Ensure your Microsoft account is an administrator account.
  2. Go to Settings > Accounts > Family & other users.
  3. Click Add account and select Create one for a child.
  4. Enter your child's email, password, name, and birthday.
  5. After setup, go to Manage family settings online or remove an account.
  6. From the online dashboard, you can:
  1. Set screen time limits.
  2. Block inappropriate websites and apps.
  3. Approve app requests.
  4. Set spending limits in the Microsoft Store. (How to Set up Parental Controls in Windows 11)

üì± iPhone: Screen Time

English Instructions

  1. Go to Settings > Screen Time.
  2. Tap Turn On Screen Time, then Continue.
  3. Select This is My Child's iPhone.
  4. Set a Screen Time passcode.
  5. Configure:
    1. Downtime.
    2. App Limits.
    3. Communication Limits.
    4. Content & Privacy Restrictions.
  6. To block websites:
  1. Go to Content & Privacy Restrictions > Content Restrictions > Web Content.
  2. Choose Limit Adult Websites or Allowed Websites Only. (Expert Tips to Monitor and Limit Screen Time on Android Phones, How to Use Parental Controls in Your Google, Apple, and Microsoft Accounts, How to Block Websites on the iPhone)
